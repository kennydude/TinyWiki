About TinyWiki looking for help?

TinyWiki is a NodeJS powered Wiki which is designed to be quite small, but with quite a few features.

Features

  • Markdown
  • Coloured Editor
  • Preview images from the editor
  • Responsive
  • Portable
  • No users, no database, no fuss *
  • Minimal
  • Simple
