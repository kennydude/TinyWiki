Limiations of this system

It's file based. No users

You cannot have any pages named as follows

  • app
  • design
  • images
  • Anything beginning with __

While nothing will break, you will only be able to edit and never ever view these pages

I want to make one

Awesome. Just add yourappname.js to the lib/app folder

Then add a page with [app yourappname]

Then write your JS file. We have jQuery already loaded (no UI)

Some things to note though:

  • #appData contains the file contents of the file you wrote. Store your data here
  • #appArea is where you should store the DOM to your app (not required, but it's for neatness)
  • When you have updated #appData with the file to replace, call saveContents() and the system will manage it all
  • If you want your app to continue showing, the first line must always be the [app yourappname] def
  • Want a lightbox? We have one showLB("html string here"); and get rid of it with hideLB();
